A simple, responsive web app that generates random colors and displays their HEX codes. Built with HTML, CSS, and vanilla JavaScript.

## Features
- Generate random colors with one click
- Displays HEX color codes
- Responsive, gradient-based UI

## Demo
Open `index.html` in your browser or use the **Live Server** extension to preview the app.

## Usage
- Click the **Generate** button to create a new palette.
- Click a HEX value to copy it (if supported).

## Development
1. Clone the repo: `git clone https://github.com/aviii-156/Random-Color-Generator-.git`
2. Open the project folder and open `index.html` in a browser or via Live Server.
3. Make changes, then refresh the page to see updates.
